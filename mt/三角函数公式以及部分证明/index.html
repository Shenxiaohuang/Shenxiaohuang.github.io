三角函数公式以及部分证明

定义

在Rt△ABCRt\triangle ABCRt△ABC中，如下有六个三角函数的定义：

正弦：

sin⁡A=ac\sin A = \frac{a}{c}sinA=ca​

级数表示：sin⁡(x)==∑k=0∞(−1)kx1+2k(1+2k)!\sin (x)==\sum_{k=0}^{\infty} \frac{(-1)^{k} x^{1+2k}}{(1+2k)!}sin(x)==∑k=0∞​(1+2k)!(−1)kx1+2k​

余弦：

cos⁡A=bc\cos A = \frac{b}{c}cosA=cb​

级数表示：cos⁡(x)=∑k=0∞(−1)kx2k(2k)!\cos (x)=\sum_{k=0}^{\infty} \frac{(-1)^{k} x^{2 k}}{(2 k) !}cos(x)=∑k=0∞​(2k)!(−1)kx2k​

正切：

tan⁡A=ab\tan A = \frac{a}{b}tanA=ba​

级数表示：tan⁡(x)=i+2i∑k=1∞(−1)kq2k for q=eix\tan (x)=i+2 i \sum_{k=1}^{\infty}(-1)^{k} q^{2 k} \color{gray}\textrm{ for } q=e^{i x}tan(x)=i+2i∑k=1∞​(−1)kq2k for q=eix

余切：

cot⁡A=ba\cot A = \frac{b}{a}cotA=ab​

级数表示：cot⁡(x)=−i−2i∑k=1∞q2k for q=eix\cot (x)=-i-2 i \sum_{k=1}^{\infty} q^{2 k} \color{gray}\text { for } q=e^{i x}cot(x)=−i−2i∑k=1∞​q2k for q=eix

正割：

sec⁡A=cb\sec A = \frac{c}{b}secA=bc​

级数表示：sec⁡(x)=−2∑k=1∞(−1)kq−1+2k for q=eix\sec (x)=-2 \sum_{k=1}^{\infty}(-1)^{k} q^{-1+2 k} \color{gray}\text { for } q=e^{i x}sec(x)=−2∑k=1∞​(−1)kq−1+2k for q=eix

余割：

csc⁡A=ca\csc A = \frac{c}{a}cscA=ac​

级数表示：csc⁡(x)=−2i∑k=1∞q−1+2k for q=eix\csc (x)=-2 i \sum_{k=1}^{\infty} q^{-1+2 k} \color{gray}\text { for } q=e^{i x}csc(x)=−2i∑k=1∞​q−1+2k for q=eix

关系 & 定理 & 公式

倒数关系

cos⁡α⋅sec⁡α=1\cos \alpha \cdot \sec \alpha = 1cosα⋅secα=1

sin⁡α⋅csc⁡α=1\sin \alpha \cdot \csc \alpha = 1sinα⋅cscα=1

tan⁡α⋅cot⁡α=1\tan \alpha \cdot \cot \alpha = 1tanα⋅cotα=1

平方关系

1+tan⁡2α=sec⁡2α1 + \tan ^ 2 \alpha = \sec ^ 2 \alpha1+tan2α=sec2α

1+cot⁡2α=csc⁡2α1 + \cot ^ 2 \alpha = \csc ^ 2 \alpha1+cot2α=csc2α

sin⁡2α+cos2α=1\sin^2 \alpha + cos ^ 2 \alpha = 1sin2α+cos2α=1

商的关系

sin⁡αcos⁡α=sec⁡αcsc⁡α=tan⁡α\frac{\sin \alpha}{\cos \alpha} = \frac{\sec \alpha}{\csc \alpha} = \tan \alphacosαsinα​=cscαsecα​=tanα

cos⁡αsin⁡α=csc⁡αsec⁡α=cot⁡α\frac{\cos \alpha}{\sin \alpha} = \frac{\csc \alpha}{\sec \alpha} = \cot \alphasinαcosα​=secαcscα​=cotα

正弦定理

asin⁡A=bsin⁡B=csin⁡C=2R=D\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R = DsinAa​=sinBb​=sinCc​=2R=D

RRR 为三角形外切圆半径，DDD 为三角形外切圆直径。

证明：

如图在 △ABC\triangle ABC△ABC 中可得 sin⁡A=hb\sin A = \frac{h}{b}sinA=bh​ 和 sin⁡B=ha\sin B = \frac{h}{a}sinB=ah​ 。

∴h=sin⁡A×b,h=sin⁡B×a ∴sin⁡A×b=sin⁡B×a ∴sin⁡Aa=sin⁡Bb ∴asin⁡A=bsin⁡B 同理：asin⁡A=csin⁡C ∴asin⁡A=bsin⁡B=csin⁡C\therefore h = \sin A \times b, h = \sin B \times a \\\ \therefore \sin A \times b = \sin B \times a \\\ \therefore \frac{\sin A}{a} = \frac{\sin B}{b} \\\ \therefore \frac{a}{\sin A} = \frac{b}{\sin B} \\\ \textrm{同理：} \frac{a}{\sin A} = \frac{c}{\sin C} \\\ \therefore \frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C}∴h=sinA×b,h=sinB×a ∴sinA×b=sinB×a ∴asinA​=bsinB​ ∴sinAa​=sinBb​ 同理：sinAa​=sinCc​ ∴sinAa​=sinBb​=sinCc​

如图， △CDB\triangle CDB△CDB 中线段 CDCDCD 经过圆心，所以 ∠CBD=90∘\angle CBD = 90 ^ \circ∠CBD=90∘ ， CD=2RCD = 2RCD=2R。

∴sin⁡A=sin⁡D=CBCD=a2R ∴asin⁡A=2R 同理：bsin⁡B=2R,csin⁡C=2R ∴asin⁡A=bsin⁡B=csin⁡C=2R=D\therefore \sin A = \sin D = \frac{CB}{CD} = \frac{a}{2R} \\\ \therefore \frac{a}{\sin A} = 2R \\\ \textrm{同理：} \frac{b}{\sin B} = 2R, \frac{c}{\sin C} = 2R \\\ \therefore \frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R = D∴sinA=sinD=CDCB​=2Ra​ ∴sinAa​=2R 同理：sinBb​=2R,sinCc​=2R ∴sinAa​=sinBb​=sinCc​=2R=D

余弦定理

a2=b2+c2−2bccos⁡A,b2=a2+c2−2accos⁡B,c2=a2+b2−2abcos⁡C 或 cos⁡A=b2+c2−a22bc,cos⁡B=a2+c2−b22ac,cos⁡C=a2+b2−c22aba ^ 2 = b ^ 2 + c ^ 2 - 2bc\cos A, b ^ 2 = a ^ 2 + c ^ 2 - 2ac\cos B, c ^ 2 = a ^ 2 + b ^ 2 - 2ab\cos C \\\ \rm{或} \\\ \cos A = \frac{b ^ 2 + c ^ 2 - a ^ 2}{2bc}, \cos B = \frac{a ^ 2 + c ^ 2 - b ^ 2}{2ac}, \cos C = \frac{a ^ 2 + b ^ 2 - c ^ 2}{2ab}a2=b2+c2−2bccosA,b2=a2+c2−2accosB,c2=a2+b2−2abcosC 或 cosA=2bcb2+c2−a2​,cosB=2aca2+c2−b2​,cosC=2aba2+b2−c2​

证明：

如图，在 △ABC\triangle ABC△ABC 中，令AB⃗=c⃗,CB⃗=a⃗,CA⃗=b⃗\vec{AB} = \vec{c}, \vec{CB} = \vec{a}, \vec{CA} = \vec{b}AB=c,CB=a,CA=b。

∴c⃗=AB⃗=CB⃗−CA⃗=a⃗−b⃗ ∴(c⃗)2=(a⃗−b⃗)2=a⃗2+b⃗2−2a⃗⋅b⃗ ∴∣c⃗∣2=∣a⃗∣2+∣b⃗∣2−2∣a⃗∣⋅∣b⃗∣⋅cos⁡C ∴c2=a2+b2−2abcos⁡C 同理：cos⁡A=b2+c2−a22bc,cos⁡B=a2+c2−b22ac\therefore \vec{c} = \vec{AB} = \vec{CB} - \vec{CA} = \vec{a} - \vec{b} \\\ \therefore (\vec{c}) ^ 2 = (\vec{a} - \vec{b}) ^ 2 = \vec{a} ^ 2 + \vec{b} ^ 2 - 2 \vec{a} \cdot \vec{b} \\\ \therefore |\vec{c}| ^ 2 = |\vec{a}| ^ 2 + |\vec{b}| ^ 2 - 2 |\vec{a}| \cdot |\vec{b}| \cdot \cos C \\\ \therefore c ^ 2 = a ^ 2 + b ^ 2 - 2ab\cos C \\\ 同理：\cos A = \frac{b ^ 2 + c ^ 2 - a ^ 2}{2bc}, \cos B = \frac{a ^ 2 + c ^ 2 - b ^ 2}{2ac}∴c=AB=CB−CA=a−b ∴(c)2=(a−b)2=a2+b2−2a⋅b ∴∣c∣2=∣a∣2+∣b∣2−2∣a∣⋅∣b∣⋅cosC ∴c2=a2+b2−2abcosC 同理：cosA=2bcb2+c2−a2​,cosB=2aca2+c2−b2​

sin⁡(α+β)=sin⁡αcos⁡β+cos⁡αsin⁡β\sin(\alpha + \beta) = \sin \alpha \cos \beta + \cos \alpha \sin \betasin(α+β)=sinαcosβ+cosαsinβ
